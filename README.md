# IOTProject

### Run 

```
git clone https://github.com/ETCxma/IOTProjet
cd IOTProjet
pip install -r requirements.txt
fastapi run main.py
```
OR
```
git clone https://github.com/ETCxma/IOTProjet
cd IOTProjet
source IOTProjet/venv/bin/activate
fastapi run main.py
```

To add random data run
`python3 remplissage.py`
